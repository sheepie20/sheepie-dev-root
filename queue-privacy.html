Privacy Policy

Effective Date: July 8, 2025

This Privacy Policy explains how Simple Queues ("the Bot") collects, stores, and uses information on Discord servers where it is installed.

1. Information Collected

The Bot stores only the minimal data required for functionality:

  • Server (guild) IDs and related configuration settings
  • Channel, category, and role IDs associated with the queueing and session system
  • Queueing history, including when users join or leave the queue
  • Information needed to manage and create temporary session voice channels
  • Logging targets (such as log channel ID)

No message content, usernames, or personally identifiable information (PII) is stored.

2. Data Usage

The stored data is used only to support the bot's features, including:

  • Automatically managing queueing behavior
  • Creating and removing temporary session voice channels
  • Moving users between channels when conditions are met
  • Logging queue events and setup activity to specified channels
  • Allowing administrators to configure the system via commands

3. Data Storage

All data is stored locally in an SQLite database file (`queueing_system.db`) on the server where the bot is hosted. No data is shared with any third parties.

4. Data Security

Standard security practices are followed to protect the database and its contents. However, no system can be guaranteed completely secure.

5. User Controls

Server administrators may use the `/reset-settings` command at any time to erase all data associated with their server, including configuration and logs.

6. Contact

This policy may be updated without prior notice. Continued use of the Bot after updates constitutes acceptance of the revised policy.
