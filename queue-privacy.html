Privacy Policy

Effective Date: July 8, 2025

This Privacy Policy explains how Simple Queues ("the Bot") handles data on Discord servers where it is installed.

1. Information Collected

The Bot stores minimal data required for functionality:

  • Server (guild) IDs and settings
  • Channel and role IDs used for queueing and sessions
  • Queueing and session history (e.g., user joins/leaves)
  • Moderation role ID and bot configuration

No user messages or personally identifiable information (PII) is stored.

2. Data Usage

The stored data is used only to provide queueing and session functionality, such as:

  • Managing session calls
  • Logging queue activity
  • Configuring server-specific settings

3. Data Storage

All data is stored locally using an SQLite database (`queueing_system.db`) on the bot host system. The data is not shared with third parties.

4. Data Security

We follow standard practices to safeguard stored settings and configuration data. However, no method of transmission or storage is 100% secure.

5. User Controls

Server administrators may use the `/reset-settings` command to erase all configuration and queueing data related to their server.

6. Contact

If you have any questions or concerns about this policy, you may contact us at sheepiegamer20@gmail.com.

This policy may be updated at any time without prior notice. Continued use of the Bot constitutes acceptance of any modifications.
